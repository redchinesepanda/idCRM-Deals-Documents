== Description ==

This is third module from id:CRM to organize deals and documents.

### Features ###
* Deals and Documents made with default WP posts so you can easily import your data from csv, xls, xml, json.
* Adaptive. Use all the functions from mobile without any applications.

* English, always included.
* Russian, always included.
